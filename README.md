## Run Locally

Clone the project

```bash
  git clone https://mr-coffee-auth-KarolinaOle
```

Go to the project directory

```bash
  cd mr-coffee-auth-KarolinaOle
```

Install dependencies

```bash
  npm install
  npm i --save-dev nodemon
```

Start the server

Terminal 1:

```
  npm run build:watch
```

Terminal 2:

```
  cd src
  npm run start:watch
```
